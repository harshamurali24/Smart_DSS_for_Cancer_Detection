# Smart_DSS_for_Cancer_Detection
## SMART DSS FOR CANCER DIAGNOSIS AND PROGNOSIS
### Early detection can save lives, but for millions of people, the barriers to timely cancer diagnosis are still very high. Advances in Al and machine learning make it possible to quickly and accurately process medical data without the need for advanced infrastructure.
### This system is designed for healthcare professionals in basic settings who need a simple and quick way to support early cancer screening.
